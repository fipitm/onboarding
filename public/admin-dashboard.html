How Submissions Work

Each time someone completes the profile form and clicks Start Planning, a new independent submission is created — regardless of which login account was used.

Scenario A — Shared session Multiple LMs sit together, use one account, make one collective decision → 1 submission representing the group.
Scenario B — Individual sessions Each LM logs in separately at their location, enters their own name and makes their own selections → separate submission per person.
